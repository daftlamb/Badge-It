How to use / 使用说明

Upload SVG / 上传 SVG
Click "upload svg" to load your SVG file. The shape will appear in 3D.
点击「upload svg」上传你的 SVG 文件，形状会以 3D 方式呈现。

Adjust / 调整参数
Use the left panel to control depth, bevel, material, color and lighting.
左侧面板可调整挤出深度、倒角、材质、颜色和灯光。

Export / 导出
Click "download png" to save a PNG screenshot.
点击「download png」保存 PNG 截图。

About Badge It

Badge It is a browser-based 3D badge / pin previewer. Upload an SVG, tweak the material and lighting, and export a PNG render — no software needed.

Badge It 是一个基于浏览器的 3D 徽章预览工具。上传 SVG，调整材质和灯光，导出 PNG 渲染图，无需安装任何软件。
